Using ASP.NET WEB API project to add/update/delete subject marks for existing students

Background of the problem statement:

Rainbow School wants to create a mobile app which will let teachers update student marks using their phones. To make that work, a WEB API has to be created which will handle web requests from the mobile application. The students and subjects database is already populated. All that is needed to add marks for subjects and students.

You must use the following:

Visual Studio 2019 Community Edition

SQL Server 17 Express Edition

Following requirements should be met:

Some of the source code should be tracked on GitHub repositories. You need to document the tracked files that are ignored during the final push to the GitHub repository.

The submission of your GitHub repository link is mandatory. In order to track your task, you need to share the link of the repository in the document.

The step-by-step process involved in completing this task should be documented.
